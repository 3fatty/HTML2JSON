# HTML2JSON
simple HTML2JSON &amp; JSON2HTML converter


##  HTML 和 JSON 的相互转换

### 主要优点:
1. 服务器端和客户端双端运行
2. JSON 数据保留 DOM 元素所有结构
3. 2JSON 和 2HTML 可配置
4. 提供 DOM 同名解析函数, 如 getElementById
